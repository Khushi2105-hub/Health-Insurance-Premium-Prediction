# Health-Insurance-Premium-Prediction
Health Insurance Premium Prediction using PySpark and Machine Learning

This project predicts health insurance charges using Machine Learning and PySpark.

## Features
- Data preprocessing using PySpark
- Feature engineering (policy_type)
- Linear Regression model
- Model evaluation (RMSE, R2)
- Data visualization

## Technologies Used
- Python
- PySpark
- Apache Spark
- Matplotlib
