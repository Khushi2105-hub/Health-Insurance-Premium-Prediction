# Health Insurance Premium Prediction using PySpark

## Project Overview
This project focuses on predicting health insurance premium charges using Machine Learning and Apache Spark (PySpark).  
It analyzes factors such as age, BMI, smoking status, and region to estimate insurance costs.

## Key Features
- Data preprocessing using PySpark
- Handling missing values
- Feature engineering (policy_type creation)
- Encoding categorical variables
- Model training using Linear Regression
- Model evaluation using RMSE and R2 score
- Data visualization

## Dataset Details
The dataset includes:
- Age
- Gender (Sex)
- BMI (Body Mass Index)
- Number of Children
- Smoking Status
- Region
- Insurance Charges (Target Variable)

## Technologies Used
- Python
- PySpark
- Apache Spark
- Matplotlib
- Seaborn

## Model Used
Linear Regression

## Results
- R2 Score ≈ 0.79
- Model performs well with some prediction variation

## Conclusion
This project shows how machine learning and PySpark can be used to predict insurance premiums effectively.
